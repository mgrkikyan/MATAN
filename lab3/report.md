#### –ê–Ω–∞–ª–∏—Ç–∏—á–µ—Å–∫–æ–µ —Ä–µ—à–µ–Ω–∏–µ 3.2

$f(x)=sinxlnx$ , $x_0=1$

–£—Ä–∞–≤–Ω–µ–Ω–∏—è –∫–∞—Å–∞—Ç–µ–ª—å–Ω–æ–π –∏ –Ω–æ—Ä–º–∞–ª–∏ –∫ –≥—Ä–∞—Ñ–∏–∫—É —Ñ—É–Ω–∫—Ü–∏–∏:

$y=f(x_0)+f'(x_0)(x-x_0)$ - —É—Ä–∞–≤–Ω–µ–Ω–∏–µ –∫–∞—Å–∞—Ç–µ–ª—å–Ω–æ–π

$y=f(x_0)-(\frac{1}{f'(x_0)})(x-x_0)$ - —É—Ä–∞–≤–Ω–µ–Ω–∏–µ –Ω–æ—Ä–º–∞–ª–∏

$f'(x)=cosxlnx+\frac{sinx}{x}$

–£—Ä–∞–≤–Ω–µ–Ω–∏—è –∫–∞—Å–∞—Ç–µ–ª—å–Ω–æ–π –∏ –Ω–æ—Ä–º–∞–ª–∏ –∫ –≥—Ä–∞—Ñ–∏–∫—É —Ñ—É–Ω–∫—Ü–∏–∏ –≤ —Ç–æ—á–∫–µ $x_0=1$:

$y=sin(1)ln(1)+(cos(1)ln(1)+\frac{sin(1)}{1})(x-1)=sin(1)(x-1)$ - —É—Ä–∞–≤–Ω–µ–Ω–∏–µ –∫–∞—Å–∞—Ç–µ–ª—å–Ω–æ–π –≤ —Ç–æ—á–∫–µ $x_0=1$

$y=sin(1)ln(1)-(\frac{1}{cos(1)ln(1)+\frac{sin(1)}{1}})(x-1)=-\frac{1}{sin(1)}(x-1)$ - —É—Ä–∞–≤–Ω–µ–Ω–∏–µ –Ω–æ—Ä–º–∞–ª–∏ –≤ —Ç–æ—á–∫–µ $x_0=1$

#### –ü—Ä–æ–≥—Ä–∞–º–º–Ω–æ–µ —Ä–µ—à–µ–Ω–∏–µ 3.2

```python
import numpy as np
import matplotlib.pyplot as plt

def f(x):
    return np.sin(x) * np.log(x)

def tangent_line(x, x0):
    return f(x0) + (np.cos(x0) * np.log(x0) + np.sin(x0) / x0) * (x - x0)

def normal_line(x, x0):
    return f(x0) - (1 / (np.cos(x0) * np.log(x0) + np.sin(x0) / x0)) * (x - x0)

x = np.linspace(0, 22, 1000)
y = f(x)
x0 = 1
y_tangent = tangent_line(x, x0)
y_normal = normal_line(x, x0)

plt.plot(x, y, label='–§—É–Ω–∫—Ü–∏—è')
plt.plot(x, y_tangent, label='–ö–∞—Å–∞—Ç–µ–ª—å–Ω–∞—è')
plt.plot(x, y_normal, label='–ù–æ—Ä–º–∞–ª—å', color='red')
plt.xlabel('X')
plt.ylabel('Y')
plt.xlim(0, 22)
plt.ylim(-15, 7)
plt.title('–ì—Ä–∞—Ñ–∏–∫ —Ñ—É–Ω–∫—Ü–∏–∏')
plt.legend()
plt.show()
```
